# 🏨 Hostel_consume_2026

A web-based Hostel Management System built using **ASP.NET Core / MVC** to manage student records, room allocation, fees, attendance, and complaints efficiently.

---

## 🚀 Features

* 👨‍🎓 Student Registration & Management
* 🛏️ Room Allocation System
* 💰 Student Fees Management
* 📅 Attendance Tracking
* 📝 Complaint Management
* 👨‍💼 Warden Management
* 🔐 Secure Login System

---

## 🛠️ Technologies Used

* **ASP.NET Core / MVC**
* **C#**
* **SQL Server**
* **Entity Framework Core**
* **HTML, CSS, JavaScript**
* **Bootstrap**

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/Hostel_consume_2026.git
```

2. Open the project in **Visual Studio**

3. Configure database in `appsettings.json`:

```
"ConnectionStrings": {
  "DefaultConnection": "Your_SQL_Server_Connection_String"
}
```

4. Run database migration:

```
Update-Database
```

5. Run the project:

```
Press F5 or Click Run
```
